Operational Docs

Step-by-step references for the four core operational areas.

📘 Doc 1: Site Operations
Beginner

How to make common updates to the site without touching code. For most changes, the answer is "edit the file in GitHub and Netlify deploys it automatically."

  • Update a product link. In GitHub, open js/main.js and find the product array. Change the link field to the new Shopify URL. Commit — live in 2 minutes.

  • Edit the homepage hero text. Open index.html in GitHub. Find the hero section (search for "Born in Montana"). Change the text. Commit.

  • Add a new recipe page. Copy an existing recipe file (e.g., lifestyle/recipes/montana-mule.html), rename it, update all the content fields, and add a redirect entry to netlify.toml. Commit both files.

  • Update business hours or contact info. Open contact.html and update the relevant field. Also update the JSON-LD block in the same file's <head> to match.

  • Replace a venue photo. Name your new WebP image with the same filename as the one you're replacing (e.g., bar.webp), upload it to the venue/ folder in GitHub, and commit. The old file is automatically replaced.

🛒 Doc 2: Shopify Operations
Core

Day-to-day Shopify tasks for running the store. Log in at montuckymoonshine.myshopify.com/admin.

  • Add a new product. Products → Add product. Fill Title, Description, Price, and Images. Set Inventory tracking on. Add to the relevant collection. Save. Then update the proposal site's js/main.js product array to reflect the new item.

  • Run a discount code. Discounts → Create discount. Choose "Percentage" or "Fixed amount." Set the code (e.g., SUMMER20), minimum order if needed, and usage limit. Save. Share the code in the email flow or social post.

  • Process a manual order. Orders → Create order. Add customer, products, and shipping. Send invoice to customer or mark as paid. Fulfill when ready.

  • Check inventory levels. Products → any product → scroll to Inventory. Adjust quantity as needed. Set a low-stock alert under Settings → Notifications.

  • Respond to an abandoned cart. Go to Orders → Abandoned checkouts. Shopify auto-sends recovery emails if configured. You can also manually send a recovery email from this view.

📱 Doc 3: Social Media Playbook
Weekly

The minimum viable social presence for Montucky — consistent posting that builds the brand without requiring a full-time social media manager.

  • Post schedule: 5 posts/week. Monday (product), Wednesday (recipe), Friday (venue/event), Saturday (lifestyle/UGC), one flexible post mid-week. Use the Social Tools page to generate captions and grab hashtag sets.

  • Content ratio: 60/30/10. 60% value content (recipes, stories, guides), 30% product/venue, 10% direct promotional. If the feed feels too "salesy," increase the value content ratio.

  • Responding to DMs and comments. Check twice daily — morning and evening. Reply within 24 hours. For venue booking requests, respond with availability and a direct contact number. For product questions, link to the Shopify product page.

  • Reposting user content. When someone tags @MontuckyMoonshine or uses #MontuckyMoonshine, save the post and request permission to share. DM: "Hey, love this — mind if we repost?" Most will say yes. Credit the original poster in the caption.

  • Monthly analytics check. Facebook Insights and Instagram Insights (built-in). Look at: reach, engagement rate, top posts. Double down on what works. Stop what doesn't within 30 days.

📅 Doc 4: Weekly Operations SOP
Recurring

Run through this every week to keep the brand operating cleanly. Total time: ~45 minutes if done consistently.

  • Monday: Content batch. Create or queue 5 posts for the week using the Social Tools page. Schedule them using Meta Business Suite or post manually.

  • Tuesday: Orders + inventory. Check Shopify for any pending orders. Fulfill outstanding items. Check inventory levels on top-selling products and reorder if needed.

  • Wednesday: Email check. Review montuckymoonshine@gmail.com for event inquiries, wholesale requests, or customer issues. Respond same day where possible.

  • Thursday: Engagement sweep. Go through Instagram and Facebook comments and DMs. Reply to any that didn't get a response. Save standout UGC for the weekend post.

  • Friday: Weekly review. Look at Shopify sales for the week. Note which products sold. Review top-performing social posts. Take 10 minutes to plan the following week's focus.

Evergreen brand awareness video. Use as profile video, pinned post, and website embed.
45–60 sec Instagram Reels / TikTok / Facebook Voiceover + B-roll

Script

Jars on the shelf. VO"Ours comes with Montana." CutProduct close-up. Logo visible. VO"Montucky Moonshine. Handcrafted goods and a private venue in Helena. "Est. Montana."

Production Notes

  • Film at golden hour for warmest light
  • Mix interior bar shots with outdoor Montana landscape
  • No talking head needed — voiceover only
  • Background music: slow, atmospheric Americana. Fade in by second sentence.
  • Caption: "Born in Montana. Made with grit. @MontuckyMoonshine #MontanaMoonshine #HelenaMT #CraftSpirits"
