# Montucky Moonshine — Proposal Site

A premium, redesigned storefront for [Montucky Moonshine](https://montuckymoonshine.com) built as a client proposal.

## Features

- **Dark rustic aesthetic** — matches the Montucky brand with elevated design
- **All products** — full catalog pulled from the Shopify store
- **Direct Shopify checkout** — every "Buy Now" button links directly to the existing Shopify product page and checkout flow
- **Collection filtering** — filter by Mugs, Tees, Merchandise
- **Fully responsive** — mobile, tablet, and desktop optimized
- **Fast** — static HTML, no framework overhead, instant load times

## Checkout Flow

All purchases route through the existing Montucky Moonshine Shopify store. When a customer clicks "Buy Now", they are directed to the Shopify product page where they can complete their purchase using the existing payment setup (Visa, Mastercard, Amex, PayPal, Apple Pay, Google Pay, Shop Pay).

## Deployment

Hosted on Netlify with automatic deploys from this repository.

## Tech Stack

- Pure HTML5, CSS3, JavaScript
- Google Fonts (Playfair Display, Barlow, Barlow Condensed)
- Deployed via Netlify
